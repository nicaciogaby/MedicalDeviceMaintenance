 Project Overview

This project aims to develop a web-based system for tracking medical device incidents and maintenance activities.

The system centralises device information, incident reports, and maintenance actions into a single application, improving visibility, traceability, and data organisation.

This project demonstrates practical skills in:

- C#
- ASP.NET Core MVC
- SQL Server
- Entity Framework Core
- Git & GitHub
- Software Architecture Principles

Project Objectives

The system allows users to:

- Register medical devices
- Log incidents related to devices
- Track maintenance actions
- Update incident status (Open / In Progress / Closed)
- Maintain a history of actions

The application follows a three-layer architecture:

- Presentation Layer (MVC Views)
- Application Layer (Controllers & Business Logic)
- Data Layer (Entity Framework Core & SQL Server)
